# Monitoring & Alerts

## Azure Monitor Metrics
- Metric monitored: CPU Percentage
- Aggregation: Average
- Time range: Last 1 hour

## Alert Configuration
- Alert name: High-CPU-Alert
- Condition: CPU Percentage > 70%
- Severity: 3
- Notification: Email alert

## Purpose
This alert helps proactively identify high CPU utilization and enables early troubleshooting before performance degradation occurs.
